# 祭壇多方塊結構擺放教學

> 本文件說明如何正確擺放儀式祭壇的多方塊結構

## 🧭 JEI 輔助

- JEI 儀式配方頁面新增「結構藍圖」區塊，預設顯示 ring1 佈局。
- 可透過按鈕切換是否顯示 ring2（對角基座）、ring3（外圈基座）的擴充建議。
- 點擊「投影」按鈕後，準星對準的方塊會生成 ghost 方塊投影，方便在世界中對齊擺放。

## 📐 結構概覽

儀式祭壇由以下方塊組成：
1. **儀式核心 (Ritual Core)** - 1 個，中心方塊
2. **奧術基座 (Arcane Pedestal)** - 至少 4 個，放置祭品
3. **魔力塔 (Mana Pylon)** - 至少 1 個，提供魔力
4. **符文石 (Rune Stone)** - 4 種各至少 1 個（效率、迅捷、穩定、增幅）
5. **粉筆符號 (Chalk Glyph)** - 至少 1 個，地板裝飾

---

## 🎯 核心擺放規則

### 1. 儀式核心位置
```
選擇一個位置放置「儀式核心」，這是整個結構的中心點。
假設核心座標為 (0, 0, 0)
```

### 2. 奧術基座（必須擺放）

**關鍵要求**：
- **必須放在核心的東西南北四個方向**
- **距離核心正好 2 格**
- **高度與核心相同（±1 格內）**
- **基座必須面向核心**

**擺放位置**（以核心為原點）：
```
俯視圖（Y 軸相同或 ±1）：

           北 (0, 0, -2)
              [基座] ← 朝南
                 ↓
西 (-2, 0, 0)  [核心]  東 (2, 0, 0)
   [基座] →    (0,0,0)   ← [基座]
                 ↑
              [基座] ← 朝北
           南 (0, 0, 2)
```

**重要**：
- 東邊的基座要**面向西**（朝核心）
- 西邊的基座要**面向東**（朝核心）
- 南邊的基座要**面向北**（朝核心）
- 北邊的基座要**面向南**（朝核心）

**放置方法**：
1. 站在核心的**東邊**，面朝**西**（看著核心），放置基座
2. 站在核心的**西邊**，面朝**東**（看著核心），放置基座
3. 站在核心的**南邊**，面朝**北**（看著核心），放置基座
4. 站在核心的**北邊**，面朝**南**（看著核心），放置基座

### 2-1. 擴充配置（可選）

為了支援需要更多祭品的儀式，祭壇可以依序擴充額外的基座環：

- **Ring 2（對角線）**：在核心對角方向（±2, 0, ±2）各放置一座基座，共 4 座。建議讓基座朝向核心，但即使保持原朝向也可啟動儀式。
- **Ring 3（外圈）**：在距離核心 4 格的東西南北方向（±4, 0, 0 與 0, 0, ±4）各放置一座基座，再增加 4 座。

這三個環加總可以提供 4、8、12 個基座供配方選擇；更多基座亦可在核心周圍（半徑 6 內、高度 ±1）按個人美觀需求加建，系統會自動計入總數。配方若標示 `pedestal.ring1` / `ring2` / `ring3` 即代表對應環的最小需求。

### 3. 魔力塔

**要求**：
- **至少 1 個魔力塔**
- **必須在核心 9 格距離內**（直線距離，不是方塊距離）
- 可以放在任意高度（在半徑範圍內）

**建議擺放位置**：
```
側視圖示例：
    [魔力塔]
       |
       |
   [核心][基座]

或者在角落：
    [魔力塔]
         \
          \
       [核心]
```

**計算公式**：
```
距離 = √((x1-x2)² + (y1-y2)² + (z1-z2)²) ≤ 9
```

### 4. 符文石

**要求**：
- **四種符文石各至少 1 個**：
  - 效率符文石 (Efficiency)
  - 迅捷符文石 (Celerity)
  - 穩定符文石 (Stability)
  - 增幅符文石 (Augmentation)
- **必須在核心 8 格範圍內**（掃描半徑）

**建議擺放**：
```
可以放在基座周圍或祭壇邊緣裝飾：

  [效率]       [迅捷]
     \           /
      [基座]-[核心]-[基座]
     /           \
  [穩定]       [增幅]
```

### 5. 粉筆符號

**要求**：
- **至少 1 個粉筆符號**
- **必須在核心水平 ±1 層，半徑 8 格範圍內**
- 有 6 種顏色與多種圖案可選
- **必須放在實體方塊上方**（粉筆是超薄覆蓋層）

**配方可能需要特定顏色或圖案組合**，例如：
- 2 個白色粉筆 + 1 個紅色粉筆
- 3 個圓形圖案

**建議擺放**：
```
地板裝飾範例（俯視圖）：

  [粉][粉][粉][粉][粉]
  [粉][基]   [核]   [基][粉]
  [粉]   [粉][粉][粉]   [粉]
  [粉][基]       [基][粉]
  [粉][粉][粉][粉][粉]
```

```
          [粉]
```

## 🚨 常見錯誤

### ❌ 錯誤 1：基座距離不對
```
[基座]     [核心]     [基座]
  ↑         ↑           ↑
距離 1 格  正確應該 2 格
```

### ❌ 錯誤 2：基座方向錯誤
```
[基座]→  →[核心]←  [基座]→
        錯誤！應該面向核心
```

### ❌ 錯誤 3：魔力塔太遠
```
                              [魔力塔]
                                 ↑
                           距離 15 格（太遠！）
       [核心]
```

### ❌ 錯誤 4：符文石種類不全
```
只放了 3 種符文石
缺少「增幅符文石」→ 驗證失敗
```

### ❌ 錯誤 5：沒有粉筆符號
```
完全沒放粉筆 → 驗證失敗
至少要 1 個粉筆符號
```

## 🎮 實際操作步驟

### 步驟 1：放置核心
1. 選擇一個平坦區域
2. 放置「儀式核心」

### 步驟 2：標記四個基座位置
1. 從核心向**東走 2 格**，標記位置
2. 從核心向**西走 2 格**，標記位置
3. 從核心向**南走 2 格**，標記位置
4. 從核心向**北走 2 格**，標記位置

### 步驟 3：放置基座（注意朝向！）
1. 站在**東側標記**，**面向核心**（朝西），放置基座
2. 站在**西側標記**，**面向核心**（朝東），放置基座
3. 站在**南側標記**，**面向核心**（朝北），放置基座
4. 站在**北側標記**，**面向核心**（朝南），放置基座

### 步驟 4：放置魔力塔
1. 在核心附近（9 格內）任意位置放置魔力塔
2. 建議放在角落或稍微高一點的位置

### 步驟 5：放置符文石
1. 在核心 8 格範圍內放置 4 種符文石各至少 1 個
2. 可以放在基座旁邊裝飾

### 步驟 6：繪製粉筆符號
1. 在地板上（核心周圍 8 格內）畫粉筆符號
2. 根據配方需求選擇顏色與圖案
3. 用同色粉筆右鍵點擊可切換圖案

### 步驟 7：啟動儀式
1. 在四個基座上放置配方需要的祭品
2. 手持催化劑（共鳴水晶或虛空珍珠）
3. 右鍵點擊儀式核心
4. 系統會自動驗證結構，如果有問題會提示錯誤訊息

## 📊 驗證系統

當你嘗試啟動儀式時，系統會檢查：

### 結構驗證
- ✅ 基座：4 個，位於東西南北，距離 2 格，面向核心
- ✅ 魔力塔：至少 1 個，距離 ≤ 9 格
- ✅ 符文石：4 種各至少 1 個
- ✅ 粉筆符號：至少 1 個

### 材料驗證
- ✅ 祭品：基座上的物品符合配方需求
- ✅ 魔力：魔力塔提供足夠的魔力
- ✅ 催化劑：使用正確的催化劑

**只有全部驗證通過，儀式才會啟動並消耗催化劑！**

## 💡 進階技巧

### 多基座配置
你可以放置**超過 4 個基座**，但至少要有東西南北四個方向各 1 個：
```
      [北基]
        ↓
[西基]→[核心]←[東基]
        ↑
      [南基]

+ 額外基座可放在對角線或其他位置
```

### 符文石堆疊
同種符文石可以放多個來增強效果：
```
效率符文石 x 3 → 效果增強
```

### 粉筆藝術
粉筆有多種顏色和圖案，可以創作美麗的地板魔法陣：
```
[白圓][白圓][白圓]
[紅三][紅五][紅三]
[白圓][白圓][白圓]
```

## 🔍 除錯指南

如果儀式無法啟動，檢查以下項目：

1. **基座數量**：至少 4 個？
2. **基座位置**：東西南北各 1 個，距離核心正好 2 格？
3. **基座朝向**：全部面向核心？
4. **基座高度**：與核心在同一水平面（±1 格內）？
5. **魔力塔距離**：至少 1 個在 9 格內？
6. **符文石種類**：4 種都有？
7. **粉筆符號**：至少有 1 個？
8. **祭品**：基座上放了配方需要的物品？
9. **催化劑**：手持正確的催化劑？
